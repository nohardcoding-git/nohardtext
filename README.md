# NoHardText

Never ship hardcoded UI strings again.

NoHardText is a localization quality platform for modern frontend projects.

It helps developers detect, understand, fix, and prevent hardcoded user-facing text before it reaches production.

## Status

Early development.

## Packages

- @nohardtext/domain
- @nohardtext/rule-engine
- @nohardtext/detect-engine
- @nohardtext/report-engine
- @nohardtext/cli

## Vision

NoHardText aims to become the ESLint of localization quality.

## Core Loop

Detect -> Understand -> Fix -> Validate -> Prevent
